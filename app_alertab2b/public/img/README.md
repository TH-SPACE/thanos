# Pasta de Imagens

Coloque aqui as imagens do sistema.

## Sugestões de Imagens

- `logo.png` - Logo do sistema
- `banner.png` - Banner para cabeçalho
- `background.jpg` - Imagem de fundo
- `grafico-exemplo.png` - Imagens para relatórios

## Como Usar

```html
<img src="/alerta-b2b/img/nome-da-imagem.png" alt="Descrição">
```

## Formatos Suportados

- **PNG** - Melhor para gráficos e imagens com transparência
- **JPG/JPEG** - Melhor para fotos
- **WEBP** - Formato moderno com melhor compressão
- **SVG** - Melhor para ícones e gráficos vetoriais

## Dicas

- Use nomes descritivos: `grafico-regionais.png` em vez de `img1.png`
- Otimize as imagens antes de subir
- Prefira tamanhos menores para melhor performance

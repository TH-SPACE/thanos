================================================================================
COMO USAR (PASSO A PASSO)
================================================================================

1. Acessar: http://localhost:3000/b2b (ou servidor)

2. Verificar permissão:
   - Botões só aparecem para usuários ADM
   - Se não aparecer, verificar perfil do usuário

3. Clicar em [📥 Sincronizar BDS] no header

4. Preencher modal:
   - Data Inicial: 01/02/2026
   - Data Final: 23/02/2026
   - Tipo de Busca: Por Data de Abertura (ou outro)
   - Regionais: Selecionar Todas (ou específicas)
   - KPIs: Automático (todos 24)

5. Clicar em [📥 Sincronizar BDS] no modal

6. Confirmar sincronização

7. Aguardar: "🔄 Sincronizando..."

8. Resultado: "✅ 85 inseridos, 15 atualizados"

9. Tabela atualizada automaticamente

================================================================================
FLUXO DE DADOS
================================================================================

FRONTEND (Modal):
  ↓
  [Data Inicial, Data Final, Tipo Busca, Regionais]
  ↓
  POST /b2b/sincronizacao/sincronizar
  ↓
BACKEND (Controller):
  ↓
  1. Autenticar na API BDS → Token
  ↓
  2. Buscar dados com filtros → Array de dados
  ↓
  3. Conectar ao MySQL
  ↓
  4. Para cada registro:
     - Mapear campos API → Banco
     - Formatar datas
     - Corrigir cluster
     - INSERT ou UPDATE
  ↓
  5. Retornar resultado
  ↓
FRONTEND:
  ↓
  Mostrar resultado e atualizar tabela

================================================================================
SOLUÇÃO DE PROBLEMAS
================================================================================

ERRO: "Falha na autenticação"
SOLUÇÃO: Verificar .env (usuário e senha corretos)

ERRO: "Nenhum dado encontrado"
SOLUÇÃO: Ampliar período ou selecionar mais regionais/KPIs

ERRO: "Timeout" ou "Conexão recusada"
SOLUÇÃO: Verificar se está na rede interna/VPN

ERRO: Botões não aparecem
SOLUÇÃO: Verificar se usuário tem perfil ADM

ERRO: Confirmação aparece antes do modal
SOLUÇÃO: ✅ JÁ CORRIGIDO - Botão header só abre modal

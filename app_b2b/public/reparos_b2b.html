Sobre o Sistema de Desempenho BDs
Visão Geral

Este sistema apresenta o desempenho dos BDs B2B Avançado, mostrando entrantes, encerramentos e eficiência por dia.

Como Funcionam os Filtros
  • Filtro de Regional: Permite selecionar uma regional específica para visualizar os dados apenas dessa região. Se nenhum filtro for aplicado, serão mostradas todas as regionais.
  • Filtro de Mês/Ano: Permite selecionar o período específico para análise. O mês atual é selecionado por padrão.
Interpretação da Tabela
  • Entrantes: Quantidade de novos registros abertos no dia (baseado na data_abertura).
  • Encerrados: Quantidade de registros encerrados no dia (baseado na data_encerramento).
  • Eficiência (%): Percentual calculado como (Encerrados / Entrantes) * 100.
  • Total: Soma acumulada para o mês inteiro.
Atualização de Dados

O indicador D-1 mostra se os dados estão atualizados até ontem (D-1) ou estão desatualizados. Clique no botão "Atualizar Dados" para carregar as informações mais recentes.
